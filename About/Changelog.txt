# Changelog for BulkRecipeGenerator

1.5.0 - Mod updated for 1.5 and passed autotests
Used DeepL to update translations for ChineseSimplified, French, German


1.4.1 - Mod updated for 1.4 and passed autotests


1.3.2 - Added unfinished thing-support for all bulk recipes. Also replaced the vague 'Default target time' option with a 'Max worktime for bulk'-option. This makes the previously hard-coded cutoff time of 120 work possible to change.


1.3.1 - Opened strings for translation, added verbose logging-option


1.0.8.0 - Fixed load order in connection to RIMMSQOL, thanks Chrispy


1.0.7.0 - Added another setting to skip making bulk recipes for quality items


1.0.6.0 - Added a setting for how quality-items are handled. Also stopped the custom recipe-view code from running when showing the health-tab.


1.0.5.0 - Products with quality should now no longer all have the same quality but be generated individually. Also a inspiration will only affect one quality-item instead of all of them


1.0.4.0 - The Limit to x5 recipes-option should now work on all recipes, but will need a restart when changed to have effect


1.0.3.0 - Mod updated for 1.3 and passed autotests


1.0.2.0 - Removed recipe icon code since that mod changed to much to modify the same way


1.0.1.0 - Updated to 1.2
